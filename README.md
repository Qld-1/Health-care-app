# Health-care-app
# Meals Page

## Description
This project contains the Meals page for the Health App.

### Features
- Navigation bar
- Page title
- Add Meal form
- Meal Name
- Calories
- Protein
- Add Meal button
- Today's Meals section with 3 sample cards
- Footer
- Basic CSS styling

## Technologies
- HTML5
- CSS3

## Git Workflow

For every task, the following workflow is used:

1. Pull the latest changes from the `main` branch.
2. Create a new feature branch (e.g. `feature/meals-page`).
3. Implement the task.
4. Create logical commits with descriptive commit messages.
5. Push the feature branch to GitHub.
6. Create a Pull Request.
7. Add at least one reviewer.
8. Fix any issues from the code review, if necessary.
9. Merge the Pull Request into `main` after approval.
